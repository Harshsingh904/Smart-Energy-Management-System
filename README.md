# Smart Energy Management System Using ANN

## Overview
This project predicts household energy consumption status (HIGH/LOW)
using an Artificial Neural Network (ANN).

## Features
- Data Cleaning
- Feature Engineering
- ANN Model
- Energy Usage Prediction
- Performance Evaluation
- ROC Curve and Confusion Matrix

## Technologies
- Python
- TensorFlow
- Keras
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Model Architecture

Input Layer (7 Features)
↓
Dense(16, ReLU)
↓
Batch Normalization
↓
Dense(8, ReLU)
↓
Dense(1, Sigmoid)

## Results
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC
